<h1> Laravel Google Sheet </h1>

# Jalankan aplikasi
- clone project
- composer install
- npm install
- ubah file .env.example menjadi .env
- buat database (sesuaikan dengan yang di .env atau sebaliknya)
- php artisan migrate
- php artisan serve
- maka aplikasi akan berjalan di localhost::8000/sheets
